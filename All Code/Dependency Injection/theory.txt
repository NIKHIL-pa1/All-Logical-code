🔷 What is Dependency Injection?

Dependency Injection is a design pattern in which an object receives its dependencies from an external source instead of 
creating them itself.

Dependency Injection is a design pattern that implements the Dependency Inversion Principle where dependencies are injected 
into a class from outside rather than the class creating them internally.
